CorvoStore

To learn more about how in-memory, key-value stores such as Redis work, I decided to build my own from scratch.

CorvoStore implements a subset of Redis functionality, including the ability to store complex data types. It is also interoperable with the Redis command-line interface. Other features it includes are maximum memory configurability, optional persistence, and a Node.js client library.
